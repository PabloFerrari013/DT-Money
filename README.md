## ðŸ”¥ How to run the project
Clone the project and access the project folder.
```bash
$ git clone https://github.com/PabloFerrari013/DT-Money
$ cd DT-Money
```

To start it, follow the steps below:
```bash
# Install as dependencies
$ yarn 

# Start the project
yarn start
```
